As we can see from the table, the dataframe consists of 344 data points. Next, we are going to construct a graph based on the data set.
<br><br>
Here's what we are going to do: we are going to plot Culmen Length (mm) against Culmen Depth (mm) for each of the datapoint, and use different colors to represent different species.
<br><br>
This can be easily achieved by the `sns.scatterplot`. Then we can set the legend and the title of the plot to make the graph more presentable.


```python
# plot all datapoints (Culmen Length against Culmen Depth)
sns.scatterplot(data = penguins,
             x = "Culmen Length (mm)",
             y = "Culmen Depth (mm)",
             hue = "Species")

# adjust the legend
plt.legend(bbox_to_anchor = (1.05, 1), loc = 2)

# set the title for the graph
plt.title(label = "Culmen Length against Culmen Depth")
```
